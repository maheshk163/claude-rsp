The organisation's current Oracle database backup architecture utilises Oracle RMAN with backups stored on NFS-mounted NAS storage. This architecture does not meet the organisation's legal and compliance requirements, as NFS-based backup storage provides no immutability or write-once protection, leaving backup data vulnerable to modification, deletion, or ransomware encryption. In addition, full database backup durations have grown to 4–6 hours or more as database sizes have increased, making backup windows operationally unsustainable and impacting production workloads. To address these requirements, the organisation has selected Cohesity DataProtect as the enterprise backup platform, which provides certified immutable backup storage through its DataLock (WORM) capability alongside significant improvements in backup performance, storage efficiency, and recovery time.
